# CogniWatch 

AI-powered early cognitive decline detection system.

## Problem
Alzheimer's disease is diagnosed too late. Early signs like sleep fragmentation, 
reduced activity, and wandering go undetected for years.

## Solution
CogniWatch monitors wearable biosensor data in real-time and uses AI to detect 
early cognitive decline before it's too late.

## Features
- Real-time patient monitoring dashboard
- AI-based cognitive health scoring (0-100)
- Automatic risk classification (Normal / Medium / High Risk)
- Live caregiver alerts
- Flask REST API backend

## Tech Stack
- Python + Flask (Backend)
- HTML + CSS + JavaScript (Frontend — single file)
- CSV-based simulated biosensor data
